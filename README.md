# 数独 / Sudoku

ブラウザで遊べる数独（ナンプレ）ゲームです。React + TypeScript + Tailwind CSS で構築されており、GitHub Pages で公開しています。

🎮 **[今すぐプレイ](https://tailofyukki-cell.github.io/sudoku-game/)**

---

## ゲーム概要

9×9のグリッドに1〜9の数字を埋めるパズルゲームです。各行・各列・各3×3ブロックに同じ数字が重複しないように配置します。毎回ランダムに生成される盤面で、何度でも新鮮な体験ができます。

**デザインコンセプト:** 和モダン × ミニマリズム。和紙色のベースに墨色のテキスト、朱色のアクセントを使った落ち着いた雰囲気のゲームです。

---

## 難易度仕様

| 難易度 | 空マス数 | 対象 |
|--------|----------|------|
| Easy   | 30〜36マス | 初心者向け |
| Normal | 40〜46マス | 標準的な難易度 |
| Hard   | 50〜56マス | 上級者向け |

すべての難易度でユニーク解（解が1つ）を保証しています。バックトラッキングアルゴリズムによる盤面生成と解の一意性検証を行っています。

## 操作方法

### マウス・タッチ操作
- セルをクリック/タップして選択
- 数字ボタン（1〜9）をクリック/タップして入力
- 削除ボタンで入力した数字を消去

### キーボード操作

| キー | 動作 |
|------|------|
| `1` 〜 `9` | 数字を入力 |
| `Delete` / `Backspace` / `0` | 数字を削除 |
| `↑` `↓` `←` `→` | セルを移動 |
| `Tab` / `Shift+Tab` | 次/前のセルへ移動 |
| `Ctrl+Z` / `Cmd+Z` | 1手戻す（Undo） |
| `M` | メモモードの切替 |

## 機能一覧

- **難易度3段階** — Easy / Normal / Hard
- **ヒント機能** — 未入力のセルに正解を1マス表示
- **Undo機能** — 直前の操作を取り消す
- **メモ機能** — セルに候補数字を小さく書き込む
- **タイマー** — プレイ時間を計測
- **ミス数カウント** — 重複入力の回数を記録
- **エラー表示** — 重複するセルを赤色でハイライト
- **関連セル強調** — 選択セルと同じ行・列・ブロックを淡色表示
- **同数字強調** — 選択セルと同じ数字のセルを強調表示
- **ベストタイム保存** — 難易度ごとのベストタイムをローカルに保存
- **途中保存・再開** — ブラウザを閉じても続きから再開可能
- **レスポンシブ対応** — スマートフォン・タブレット・PCに対応

---

## 盤面生成ロジック

1. **完成盤面の生成**: バックトラッキングアルゴリズムでランダムな完成盤面を生成
2. **マスの削除**: ランダムな順序でマスを削除し、削除のたびに解の一意性を検証
3. **ユニーク解保証**: 解が2つ以上になる場合はそのマスを復元し、別のマスを試みる
4. **難易度調整**: 難易度ごとの目標空マス数に達するまで繰り返す

## ローカル実行方法

```bash
# リポジトリをクローン
git clone https://github.com/tailofyukki-cell/sudoku-game.git
cd sudoku-game

# 依存パッケージをインストール
pnpm install

# 開発サーバーを起動
pnpm dev
```

ブラウザで `http://localhost:5173/sudoku-game/` を開いてください。

### ビルド

```bash
pnpm build
```

`dist/` ディレクトリに静的ファイルが生成されます。
